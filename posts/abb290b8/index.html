Firefox便捷版制作

发表于 2021-05-20 分类于 工具 阅读次数：

配置文件

Windows

  • 地址栏输入 about:profiles 打开配置文件保存目录，存有 .default 后缀的目录为配置文件目录。

  • 也可直接访问，%APPDATA%\Mozilla\Firefox\Profiles\ 进入配置文件保存目录。

Mac OS

Library (资源库) 下的 Application Support 文件夹 >> Firefox 文件夹 >> Profiles >> 以 .default

配置文件信息

  • places.sqlite : 书签、下载历史和浏览历史，bookmarkbackups 文件夹中存储着书签备份文件，可用于恢复书签
  • key3.db 、 logins.json : 密码
  • permissions.sqlite 、content-prefs.sqlite : 站点自定的首选项
  • search.jsom.mozlz4 : 搜索引擎
  • persdict.dat : 用户词典
  • formhistory.sqlite 自动完成历史
  • cookies.sqlite : Cookie
  • DOM 存储: 比 Cookie 更强大安全易用的一种存储信息方式，每个站点的信息都被存储在 webappsstore.sqlite 文件中，而 chromeappstore.sqlite 存储着 about:* 页面的信息
  • extensions : 扩展
  • cert8.db : 安全证书设置
  • secmod.db : 安全设备设置
  • mimeTypes.rdf : 下载动作
  • pluginreg.dat 插件 MIME 类型
  • sessionstore.js : 保存的会话
  • xulstore.json : 指导员工具栏
  • prefs.js : 用户首选项

这些文件会随着用户在火狐中的操作同步修改。

通过使用参数选项 -profile 设置使用启用的配置

  • Windows 中可在快捷方式的 属性 里的 目标 最后添加参数,如 "C:\Program Files (x86)\Mozilla Firefox\firefox.exe" -profile "D:\Profile"
  • Mac OS 可使用命令 /Applications/Firefox.app/Contents/MacOS/firefox-bin -profile /home/Profile

制作便捷版火狐

  • 将火狐安装目录复制出来
  • 将 Profiles 下的配置文件夹 (.default 结尾的目录) 复制到刚才复制出来的目录中，并命名为 Profile

  • 创建启动脚本

    • Windows : start "" "firefox.exe" -profile "Profile" >> run.bat
    • MacOS : ./firefox-bin -profile ./Profile >> run.sh

目录结构

|-Firefox/
|-|-firefox.exe(firefox-bin)
|-|-Profile/
|-|-run.bat(run.sh)
